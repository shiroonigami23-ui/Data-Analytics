ℹ️ About This Project

The Data Analytics Hub is built for 3rd year B.Tech students (and beyond!) to make learning data analytics more fun, interactive, and effective.

✨ Why this Hub?

  • 📚 Auto-extract summaries and visuals from uploaded PDFs, DOCX, PPTX, and images.
  • 📝 Generate practice quizzes based on real resources.
  • 🏆 Gamified learning with badges, progress tracking, and streaks.
  • 🌐 Translator integration for quick word meaning.
  • 📊 Analytics dashboard for site usage.

🛠️ Tech Stack

Frontend: HTML, CSS (enhanced), JavaScript (dynamic updates)
Backend Automation: Python (PyPDF2, pdfminer, docx, pptx)
CI/CD: GitHub Actions for auto-update and deploy
Feedback: Google Apps Script → Sheets + email summary
